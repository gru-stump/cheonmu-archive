# 세계관 기록실 메뉴 설계

## 목표

기록철 CM-01~CM-06에서 독자가 확인할 수 있는 정보로 별도의 `세계관` 메뉴를 만든다. 페이지는 특재청 내부 기록실처럼 보이되, 어려운 설정에는 짧은 일반 설명을 병기한다. 천령의 정체와 능력 원리는 직접 밝히지 않고 관측 단서와 잠긴 기밀 문서로만 제시한다.

## 공개 기준

- 현재 공개 단계는 `CM-06`이다.
- 공개 문서에는 CM-01~CM-06의 요약과 승인된 장면에서 확인 가능한 사실만 사용한다.
- 천령을 `인외`라고 직접 명명하지 않는다.
- 낮은 체온, 불안정한 측정값, 설명하기 어려운 치료 반응은 관측 사실로만 기록한다.
- 천령의 피의 성질, 실제 나이와 기원, 특재청 잔류 이유, 능력의 원리와 한계는 공개하지 않는다.
- 미확정 canon을 설명 편의를 위해 새로 확정하지 않는다.
- 이후 기록철이 추가되면 문서별 공개 단계 값만 올려 잠긴 단서를 순차적으로 해금할 수 있어야 한다.
- 기존 `천무 설정`, 천령 전체 프로필, 관계 전체 프로필과 CM-06 원문은 공개 컬렉션 밖에 원형 그대로 보관한다.
- 공개 천령 프로필과 관계 문서는 CM-06 시점에서 관측·확인 가능한 정보만 남긴 별도 공개본으로 교체한다.

## 내비게이션과 주소

상단 내비게이션을 다음 순서로 구성한다.

`천무 · 기록철 · 세계관 · 아카이브`

- 세계관 첫 화면: `#/world`
- 개별 문서: `#/world/:documentId`
- 존재하지 않는 문서 ID는 세계관 첫 문서로 안전하게 대체한다.
- 관련 기록 표식은 기존 `#/records/:recordId` 상세 페이지로 연결한다.

## 화면 구조

### 데스크톱

- 왼쪽에는 분류 색인을 배치하고 오른쪽에는 선택한 문서를 표시한다.
- 색인은 문서를 바꿔도 같은 위치에 유지한다.
- 선택된 분류와 문서는 시각적·접근성 상태로 함께 표시한다.

### 모바일

- 문서 상단에 분류 선택 버튼을 둔다.
- 버튼을 누르면 색인이 펼쳐지고, 문서를 선택하면 열람 영역으로 이동한다.
- 키보드와 터치 모두에서 문서 선택과 관련 기록 이동이 가능해야 한다.

### 시각 언어

- 기존 사이트의 검정색 기록철·기관 문서 분위기를 유지한다.
- 문서 번호, 열람 등급, 기준 시점, 관측 상태를 메타데이터로 표시한다.
- 본문은 소설 리더보다 조밀한 문서형 크기와 행간을 사용한다.
- 기밀 영역은 자물쇠, 검열선, 낮은 대비로 표현하되 공개 단서는 정상적으로 읽을 수 있어야 한다.
- 잠긴 부분에는 미래 단계나 사건명을 노출하지 않고 `추가 기록 확인 후 해금`이라고 표시한다.

## 분류 색인

1. 기관과 조직
2. 현장 대응
3. 의료와 치료
4. 미확인 개체·오염
5. 관측 기록
6. 기밀 문서

## 초기 문서

### WF-01 특수재난관리청

- 분류: 기관과 조직
- 상태: 공개
- 내용: 미확인 개체와 특수재난을 관리하는 기관. 현장 통제, 격리, 구조, 의료 지원을 담당한다.
- 일반 설명: 현장 부서와 의료 부서를 함께 운용하는 특수재난 대응 기관이다.
- 관련 기록: CM-01, CM-03

### WF-02 특수기동대

- 분류: 기관과 조직
- 상태: 공개
- 내용: 고위험 현장에 투입되는 대응 부서. 지휘관은 철수 순서와 구조 인원의 안전을 확인한다. 무영이 현장 지휘를 맡고 있다.
- 일반 설명: 무영이 소속된 현장 대응 부서다.
- 관련 기록: CM-01, CM-03

### WF-03 제3의료센터

- 분류: 의료와 치료
- 상태: 공개
- 내용: 특수재난 부상자와 오염 노출자를 치료·관찰하는 의료 기관. 천령이 의사이자 연구원으로 소속되어 있다.
- 일반 설명: 천령이 근무하며 무영의 치료가 반복되는 곳이다.
- 관련 기록: CM-01, CM-02

### WF-04 격리 및 철수 절차

- 분류: 현장 대응
- 상태: 공개
- 내용: 미확인 개체가 관측된 현장은 격리선을 기준으로 통제된다. 의료진도 철수 명령의 대상이지만 현장 판단과 충돌한 사례가 존재한다.
- 일반 설명: 현장 인원을 살리기 위한 통제 절차이며, 두 사람의 판단이 자주 부딪치는 지점이다.
- 관련 기록: CM-01, CM-03

### WF-05 오염과 치료

- 분류: 미확인 개체·오염, 의료와 치료
- 상태: 일부 공개
- 내용: 오염은 일반 외상과 구분해 다뤄진다. 노출 정도에 따라 장비 측정과 의료 관찰이 이루어진다. 치료 방식과 일부 회복 사례는 제한 정보다.
- 일반 설명: 특수재난 현장의 부상은 상처만 치료한다고 끝나지 않는다.
- 관련 기록: CM-01, CM-05

### OB-01 반복 관측 이상

- 분류: 관측 기록
- 상태: 일부 공개
- 내용: 천령 주변에서 낮은 체온, 불안정한 측정값, 설명하기 어려운 치료 반응이 반복된다. 원인은 확정하지 않는다.
- 공개 단서: CM-01, CM-03, CM-05에서 관측된 현상만 연결한다.
- 일반 설명: 서로 다른 현장에서 같은 종류의 이상이 반복되고 있다.
- 관련 기록: CM-01, CM-03, CM-05

### CL-01 의료인 천령 관련 제한 기록

- 분류: 기밀 문서
- 상태: 잠김
- 현재 공개 문구: `신원 기록 일부 불일치`, `특정 오염 상황에서 비정상 반응 관측`
- 잠금 문구: `추가 기록 확인 후 해금`
- 공개 금지: 정체의 직접 명칭, 피의 성질, 실제 나이와 기원, 특재청 잔류 이유, 능력의 원리와 한계
- 관련 기록: CM-01, CM-05

## 기존 공개 자료의 기밀 정리

세계관 메뉴의 잠금 연출이 기존 아카이브에서 우회되지 않도록 공개 자료를 같은 CM-06 기준으로 정리한다.

### 천무 설정 문서

- 현재 `src/content/documents/settings.md` 원문은 공개 문서 glob이 읽지 않는 비공개 하위 폴더로 이동한다.
- 아카이브 카드와 직접 문서 주소에서 더 이상 노출하지 않는다.
- 공개 가능한 기관·현장·의료 정보는 세계관 문서로만 제공한다.

### 천령 프로필

- 현재 전체 프로필 원문은 비공개 하위 폴더에 보관한다.
- 공개 프로필에는 이름, 외관상 연령대, 성별, 신장, 직업, 소속, 외형, 관찰 가능한 성격과 말투를 유지한다.
- 낮은 체온과 설명하기 어려운 이질감은 관측 표현으로만 유지할 수 있다.
- 종족의 직접 명칭, 실제 나이, 동물화, 피의 성질, 능력 원리·한계, 오염 내성은 삭제한다.

### 관계 프로필

- 현재 전체 관계 원문은 비공개 하위 폴더에 보관한다.
- 공개본에는 첫 조우부터 CM-06까지의 관계 흐름, 서로를 바라보는 시선, 호칭과 관계 변화만 유지한다.
- `인외 의사`, `독과 약`, 능력의 직접 설명과 CM-07 이후의 `귀환의 약속` 표현은 삭제하거나 CM-06 이전 표현으로 교체한다.

### CM-06

- 현재 원문은 비공개 기록으로 보관한다.
- 공개 요약의 `인외` 직접 명칭은 `통제 밖으로 벗어나는 것을 견디지 못하는 치료자`처럼 관찰 가능한 관계 표현으로 교체한다.
- 대사와 사건 단계는 바꾸지 않는다.

## 데이터 구조

세계관 문서는 기존 일반 설정 문서와 분리된 전용 콘텐츠 컬렉션으로 관리한다. 각 문서는 다음 정보를 제공한다.

- 안정적인 문서 ID와 문서 번호
- 제목과 분류
- 기관 문서 본문과 일반 설명
- 관련 기록 ID 목록
- 최초 공개 단계와 단서별 공개 단계
- 열람 상태: 공개, 일부 공개, 잠김

현재 공개 단계는 한 곳에서 `6`으로 관리한다. 화면은 단계 이하의 내용만 렌더링하고, 아직 공개할 수 없는 내용은 원문 자체를 공개 번들에 넣지 않거나 공개용 잠금 문구로 대체한다. 미래의 비밀 원문을 CSS로만 가리는 방식은 사용하지 않는다.

## 오류와 안전성

- 관련 기록 ID가 실제 공개 기록과 연결되지 않으면 콘텐츠 검증이 실패해야 한다.
- 중복 세계관 문서 ID와 문서 번호를 거부한다.
- 존재하지 않는 문서 경로는 첫 공개 문서를 표시한다.
- 잠긴 정보의 실제 원문은 DOM이나 공개 JavaScript 번들에 포함하지 않는다.
- 모든 비공개 하위 폴더는 Vite의 직접 하위 파일 glob, CLI 검증과 편집기 공개 콘텐츠 조립에서 제외한다.
- `_hidden` 기록과 사용자 원본 PDF·이미지·프로필 문서는 세계관 공개 데이터로 자동 유입하지 않는다.

## 테스트 범위

- 상단에 `세계관` 메뉴가 올바른 순서로 표시된다.
- `#/world`와 개별 문서 주소가 정상 동작한다.
- 데스크톱 색인과 모바일 펼침 색인이 문서를 변경한다.
- 공개 문서가 CM-01~CM-06 범위의 내용과 관련 기록 링크를 표시한다.
- 기밀 문서는 승인된 두 단서와 잠금 문구만 표시한다.
- 금지된 직접 명칭과 비밀 원문이 DOM과 프로덕션 번들에 없다.
- 공개 아카이브에서 `천무 설정` 문서가 사라지고, 공개 천령·관계 프로필과 CM-06에 금지 표현이 없는지 검증한다.
- 비공개 원본 파일은 보존되지만 프로덕션 번들에는 포함되지 않는지 고유 문구로 검증한다.
- 잘못된 경로의 안전한 대체, 키보드 조작, 현재 선택 상태를 검증한다.
- 기존 기록철, 장면 재구성, 아카이브 경로의 회귀가 없어야 한다.

## 범위 밖

- CM-07 이후 기록의 공개
- 천령의 정체·피·기원·잔류 이유에 대한 확정 설명
- 관리자 화면에서 공개 단계를 조절하는 기능
- 새 설정을 canon으로 확정하는 작업
- 배포는 구현과 검증 후 사용자가 명시적으로 요청할 때만 수행한다.
