# 비밀 공개 계획

| 비밀 | 초기 단서 | 다음 단계 | 확인 gate |
| --- | --- | --- | --- |
| Cheonryeong not ordinary | 낮은 체온, 비정상적으로 고요한 반응, 일관되지 않은 측정값 | 불가능해 보이는 회복 또는 오염 반응의 부분 목격 | 사용자가 승인한 확인 에피소드 |
| blood heals and harms | 냄새, 주변 인물의 반응, 감춘 처치법, 대가 | 모호하게 보이는 사용 장면 | 별도의 기제와 한계 승인 |
| age/origin | 빠진 기록, 회피하는 날짜, 시대에 맞지 않는 지식 | 서로 모순되는 기관 기록 | 사용자가 canon을 확정 |
| reason for staying | 회피, 이례적인 합의 | 부분적인 거래 또는 의무 | 사용자가 동기를 확정 |

독자는 첫 번째 비밀을 무영보다 먼저 추론할 수 있다. 그러나 어느 행도 하나의 설명 장면으로 합치지 않는다. 피, 나이와 기원, 잔류 이유의 공개는 각각 독립된 gate를 통과한다.
