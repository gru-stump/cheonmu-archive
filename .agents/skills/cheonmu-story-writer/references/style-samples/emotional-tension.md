# 감정 긴장

| 필드 | 내용 |
| --- | --- |
| 좋아하는 부분 | 말하지 않은 감정이 거리, 침묵, 멈칫거림으로 화면에 남는 방식 |
| 천무에 적용할 부분 | 긴장은 두 사람 사이의 물리적 거리와 시선의 방향으로 먼저 표현한다. 천령은 여유가 무너지는 순간(웃음이 사라짐, 말이 짧아짐)으로, 무영은 습관의 어긋남(출입구 확인을 잊음, 치료 후 바로 일어나지 않음)으로 긴장을 드러낸다. 감정의 이름은 장면이 끝난 뒤에도 붙이지 않는다. |
| 피할 부분 | '그는 불안했다' 같은 직접 명명, 감정 고조를 위해 인물이 평소 성격과 다르게 행동하는 전개 |
| 중요도 | 높음 |
| 참고 내용/분석 메모 | 표현 원리만 제공하는 non-canon 문체 자료다. 사건, 대사, 관계 단계의 사실 근거로 사용하지 않는다. |
