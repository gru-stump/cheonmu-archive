# 절제된 로맨스

| 필드 | 내용 |
| --- | --- |
| 좋아하는 부분 | 고백보다 시선, 거리, 치료 후 남은 동작으로 감정을 드러내는 방식 |
| 천무에 적용할 부분 | 평상시에는 사건과 행동을 선명하게 쓰고, 관계의 거리가 실제로 바뀌는 순간에만 감각 묘사의 밀도를 높인다. |
| 피할 부분 | 한 번의 대화로 갈등을 해소하거나 관계 단계를 건너뛰는 직접 고백 |
| 중요도 | 높음 |
| 참고 내용/분석 메모 | 표현 원리만 제공하는 non-canon 문체 자료다. 사건, 대사, 관계 단계의 사실 근거로 사용하지 않는다. |
