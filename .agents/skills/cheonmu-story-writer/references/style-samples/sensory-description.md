# 감각 묘사

| 필드 | 내용 |
| --- | --- |
| 좋아하는 부분 | 감정을 설명하는 대신 체온, 냄새, 통증, 소리 같은 몸의 감각으로 장면의 상태를 전달하는 방식 |
| 천무에 적용할 부분 | 치료 장면에서는 천령의 차가운 손끝, 피와 소독약 냄새, 무영의 억누른 호흡처럼 POV 인물이 실제로 느낄 수 있는 감각 한두 개에 집중한다. 같은 감각(체온 차, 피 냄새)을 관계 단계가 바뀔 때마다 다른 의미로 재사용해 변화를 드러낸다. |
| 피할 부분 | 한 문단에 여러 감각을 나열하는 카탈로그식 묘사, POV 인물이 알 수 없는 상대의 내부 감각을 단정하는 문장 |
| 중요도 | 높음 |
| 참고 내용/분석 메모 | 표현 원리만 제공하는 non-canon 문체 자료다. 사건, 대사, 관계 단계의 사실 근거로 사용하지 않는다. |
