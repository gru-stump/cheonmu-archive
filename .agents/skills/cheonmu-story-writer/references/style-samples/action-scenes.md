# 액션 장면

| 필드 | 내용 |
| --- | --- |
| 좋아하는 부분 | 짧은 문장과 명확한 공간 배치로 속도를 만들고, 부상이 장면 이후에도 비용으로 남는 방식 |
| 천무에 적용할 부분 | 현장 장면은 무영의 판단 순서(위험 파악 → 대원 → 자신)를 문장 순서로 옮긴다. 타격과 부상은 감각 한 개(소리, 충격, 열)로 처리하고, 통증의 실감은 현장이 끝난 뒤 치료 장면에서 청구한다. 천령의 개입은 조용하고 매끄러운 움직임으로, 전투력의 과시 없이 결과만 보여준다. |
| 피할 부분 | 초 단위 실황 중계식 서술, 부상을 입고도 다음 장면에서 아무 제약 없이 움직이는 전개 |
| 중요도 | 보통 |
| 참고 내용/분석 메모 | 표현 원리만 제공하는 non-canon 문체 자료다. 사건, 대사, 관계 단계의 사실 근거로 사용하지 않는다. |
