# 대사 리듬

| 필드 | 내용 |
| --- | --- |
| 좋아하는 부분 | 말수와 문장 길이의 비대칭으로 두 인물의 성격 차이가 대화 자체에서 드러나는 방식 |
| 천무에 적용할 부분 | 천령은 길게 늘이고 덧붙이며 떠보고, 무영은 짧게 자르고 필요한 말만 한다. 이 비대칭을 유지하되, 위기에서는 역전시킨다 — 천령이 짧아지고 무영이 약속의 문장을 늘인다. 대사 사이의 행동 지문은 최소한으로 두고, 침묵도 대답으로 취급한다. |
| 피할 부분 | 두 인물의 말투가 비슷해지는 긴 설명 대화, 정보 전달을 위해 인물이 이미 아는 것을 서로 말해주는 대사 |
| 중요도 | 높음 |
| 참고 내용/분석 메모 | 표현 원리만 제공하는 non-canon 문체 자료다. 사건, 대사, 관계 단계의 사실 근거로 사용하지 않는다. |
