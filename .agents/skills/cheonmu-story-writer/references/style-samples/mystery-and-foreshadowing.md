# 미스터리와 복선

| 필드 | 내용 |
| --- | --- |
| 좋아하는 부분 | 단서가 장면의 자연스러운 일부로 놓여 있다가, 공개 후 앞 장면의 의미가 바뀌는 방식 |
| 천무에 적용할 부분 | 천령의 비범함에 관한 단서는 항상 그럴듯한 표면 설명과 함께 둔다(낮은 체온 → 원래 손이 찬 사람, 태연함 → 의사의 직업적 침착). 같은 단서를 최소 세 가지 다른 기능(치료 장면의 감각, 기록의 어긋남, 타인의 반응)으로 변형해 반복한다. 공개 순서와 gate는 `reveal-plan.md`가 정본이다. |
| 피할 부분 | 단서를 강조하는 서술자의 눈짓('그때는 몰랐다' 류), 공개 장면에서 모든 비밀을 한꺼번에 설명하는 lore recital |
| 중요도 | 높음 |
| 참고 내용/분석 메모 | 표현 원리만 제공하는 non-canon 문체 자료다. 사건, 대사, 관계 단계의 사실 근거로 사용하지 않는다. |
