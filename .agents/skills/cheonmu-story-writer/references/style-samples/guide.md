# 문체 자료 안내

다음 파일이 준비되어 있다. 필요하면 같은 양식으로 갱신하거나 추가한다.

- `sensory-description.md`
- `emotional-tension.md`
- `dialogue-rhythm.md`
- `action-scenes.md`
- `restrained-romance.md`
- `mystery-and-foreshadowing.md`

이 폴더의 PDF 작법 자료는 성인 수위가 승인된 작업에서만 참조한다(SKILL.md `수위 운용` 참조). 기본 수위 작업에서는 열지 않는다.

각 자료는 아래 필드를 사용한다.

| 필드 | 내용 |
| --- | --- |
| 좋아하는 부분 | 유지하고 싶은 표현의 효과 |
| 천무에 적용할 부분 | 이 작품의 장면에 옮길 수 있는 높은 수준의 기법 |
| 피할 부분 | 반복하거나 과장하지 않을 요소 |
| 중요도 | 낮음, 보통, 높음 |
| 참고 내용/분석 메모 | 필요한 발췌와 그 효과의 분석 |

본인 글은 길게 넣어도 된다. 다른 작가의 글은 꼭 필요한 짧은 발췌와 분석만 둔다. 모든 문체 자료는 canon이 아니다.
